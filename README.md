# Global Education Data Analysis (EDA)

## Overview
This project focuses on Exploratory Data Analysis (EDA) of global education and literacy data. The analysis explores literacy rates across different education levels, gender-based literacy differences, and trends in educational development across countries and regions.

The goal of this project is to uncover meaningful insights from education-related datasets using Python data analysis and visualization libraries.

## Objectives
- Analyze global literacy rates
- Compare male and female literacy levels
- Study lower and upper education literacy trends
- Identify gender gaps in education
- Visualize country-wise and region-wise educational patterns
- Understand relationships between literacy indicators

---

## Dataset Features
The dataset includes variables such as:
- Country/Region
- Overall Literacy Rate
- Male Literacy Rate
- Female Literacy Rate
- Lower Education Literacy
- Upper Education Literacy
- Education Enrollment Indicators
- Gender-based educational statistics

## Key Insights
- Literacy rates vary significantly across regions
- Gender gaps are more visible in certain countries
- Female literacy rates show improvement in many developing regions
- Higher education literacy generally correlates with overall literacy growth
